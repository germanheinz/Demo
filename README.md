# SpringBoot

Minimal Spring Boot 3.5.3 starter project for coding practice.

## Stack
- Java 17
- Spring Boot 3.5.3
- Spring Web
- Spring Data JPA + H2 (in-memory)
- Bean Validation
- Lombok
- MapStruct
- JUnit 5 + Mockito + MockMvc + AssertJ

## Run
```bash
mvn spring-boot:run
```

## Test
```bash
mvn test
```

## H2 Console
http://localhost:8080/h2-console  
JDBC URL: `jdbc:h2:mem:devdb`
